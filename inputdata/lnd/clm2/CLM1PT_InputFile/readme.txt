This folder contains files for creation of driving flies for CLM point version model simualtion. 
The matlab code was programmed by Fengming Yuan at ORNL, now archived at EMAIL lab at SDSU.
1) create ASCII file with all variables needed 
time,ZBOT,WIND,PSRF,TBOT,RH,PRECTmms,FSDS,FLDS
2) copy the ASCII file into monthly_txt folder
3) revise the start year and end year in the file clm_climatenc.m
4) execute the program within matlab
5) check out the netcdf file in the monthly_nx folder.
